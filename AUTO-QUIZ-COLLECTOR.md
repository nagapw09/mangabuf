# 🤖 Автоматический сборщик вопросов квиза

## 📝 Что делает расширение?

Когда вы включаете **Квиз**, расширение:

1. ✅ **Читает вопрос** со страницы квиза
2. 🔍 **Проверяет** есть ли он в базе `quiz-answers.json`
3. 📝 **Если НЕТ в базе:**
   - Записывает вопрос как новый
   - Пробует угадать ответ (случайный выбор)
   - После ответа проверяет правильный или нет
   - Если правильный - **автоматически сохраняет**
   - Если неправильный - запоминает и попробует другой
4. ✅ **Если ЕСТЬ в базе:**
   - Использует известный правильный ответ

## 🚀 Как пользоваться?

### 1. Запустите квиз
```
1. Откройте MangaBuff Helper
2. Нажмите "Старт" в секции "Квиз"
3. Расширение начнет проходить квиз
```

### 2. Квиз автоматически:
- Отвечает на вопросы из базы ✅
- Собирает НОВЫЕ вопросы 📝
- Определяет правильные ответы 🎯
- Сохраняет их автоматически 💾

### 3. Просмотр собранных вопросов

В popup появилась новая секция **"Собранные вопросы"**:

```
📝 Собранные вопросы
Вопросы, которые квиз автоматически собрал

✅ Автосбор: квиз записывает новые вопросы и ищет правильные ответы

[👁️ Просмотр]  [📥 Экспорт]  [🗑️ Очистить]
```

**Кнопки:**
- **👁️ Просмотр** - показать все собранные вопросы и скопировать в буфер
- **📥 Экспорт** - скачать JSON файл с вопросами
- **🗑️ Очистить** - удалить все собранные вопросы

## 📊 Что сохраняется?

Для каждого вопроса сохраняется:
```json
{
  "question": "Текст вопроса?",
  "answers": ["Вариант 1", "Вариант 2", "Вариант 3", "Вариант 4"],
  "correct_text": "Правильный ответ",
  "timestamp": 1234567890,
  "source": "auto-collected"
}
```

## 🔄 Как это работает?

### Сценарий 1: Вопрос уже в базе
```
1. Квиз показывает вопрос
2. Расширение находит его в quiz-answers.json
3. Выбирает правильный ответ ✅
4. Переходит к следующему вопросу
```

### Сценарий 2: Новый вопрос
```
1. Квиз показывает вопрос
2. Расширение НЕ находит его в базе ❌
3. Записывает вопрос и все варианты ответов 📝
4. Выбирает случайный ответ 🎲
5. После клика проверяет результат:
   - Если ✅ правильно → сохраняет правильный ответ
   - Если ❌ неправильно → запоминает и попробует другой
6. Квиз начинается заново (при неправильном ответе)
7. Расширение повторяет попытки пока не найдет правильный
```

## 💾 Где хранятся вопросы?

1. **Временно** - в памяти браузера (Chrome Storage)
2. **Постоянно** - экспортируйте кнопкой "📥 Экспорт"

Экспортированный файл будет называться:
```
mangabuff-questions-1234567890.json
```

## 🔧 Технические детали

### Определение правильности ответа

После клика расширение ищет элемент с результатом:
```javascript
const resultEl = document.querySelector('.quiz__result, [class*="quiz-result"]');
const isCorrect = resultText.includes('правильно') || resultText.includes('верно');
```

### Сохранение попыток

Каждая попытка ответа сохраняется:
```javascript
attempts: [
  { answer: "Саске", isCorrect: false },
  { answer: "Наруто", isCorrect: true }
]
```

### Отправка в background

Когда найден правильный ответ:
```javascript
chrome.runtime.sendMessage({
  action: 'saveNewQuestion',
  data: {
    question: "Как зовут главного героя?",
    answers: ["Наруто", "Саске", "Какаши", "Итачи"],
    correct_text: "Наруто"
  }
});
```

## 📋 Формат экспорта

Файл содержит массив вопросов:
```json
[
  {
    "question": "Как зовут главного героя?",
    "answers": ["Наруто", "Саске", "Какаши", "Итачи"],
    "correct_text": "Наруто",
    "timestamp": 1730000000000,
    "source": "auto-collected"
  },
  ...
]
```

Этот файл можно:
1. Открыть в любом текстовом редакторе
2. Использовать для обновления `quiz-answers.json`
3. Поделиться с другими пользователями

## ⚙️ Настройки

Квиз работает с настройками:
- **Думает 2-5 секунд** перед ответом (выглядит естественно)
- **Ждет 1.5-2.5 секунды** после клика (проверяет результат)
- **Автоматически начинает заново** при неправильном ответе

## 🎯 Преимущества

✅ **Автоматический сбор** - не нужно вручную записывать  
✅ **Определение правильных ответов** - пробует пока не найдет  
✅ **Без дубликатов** - проверяет что вопрос еще не собран  
✅ **Экспорт в JSON** - удобный формат для работы  
✅ **Логи в консоли** - видно что происходит  

## 🚨 Важно!

- Квиз **перезапускается** при неправильном ответе
- Расширение **автоматически** попробует снова
- Может потребоваться **несколько попыток** для нового вопроса
- Все попытки **логируются** в консоли

## 📊 Статистика

Счетчик в секции показывает:
```
📝 Собранные вопросы [5] ← количество собранных
```

Обновляется автоматически каждые 5 секунд.
