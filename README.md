# ⛏️ MangaBuff Helper

Расширение для автоматизации на mangabuff.ru: шахта, комментарии, читалка манги.

## 🚀 Быстрая установка

1. Открой `chrome://extensions/`
2. Включи "Режим разработчика"
3. "Загрузить распакованное расширение"
4. Выбери папку `extension`
5. Готово!
- Зайди на mangabuff.ru
- Нажми на иконку расширения
- Пользуйся!

## 🎯 Что умеет

### ⛏️ Шахта (Mine)
- Авто-клики с задержкой 300-800мс
- Счетчик кликов
- Лимит (или бесконечно)

### 💬 Комментарии
- 10 или 13 комментов (переключатель)
- Случайные тексты
- Задержка 1-2 секунды

### 📖 Читалка
- Автопрокрутка вниз
- Скорость 1-10
- **Работает только на страницах манги** (`mangabuff.ru/manga/...`)
- **💰 БОЛЬШАЯ тыква** (`event-bag`) → кликает пока не пропадет (задержка 1000мс)
- **🎃 Маленькая тыква слева** → 1 клик
- **После тыквы:** ждет 5 минут, переходит на след. страницу
- **Лимит:** 60 тыкв в день (автосброс в 23:00, как на сервере MangaBuff)
- **Автовосстановление:** продолжает работу после перехода на новую страницу

### 🎯 Квиз + 🤖 Автосбор вопросов
- Автоматическое прохождение квиза
- База с 421 правильными ответами
- **🆕 АВТОМАТИЧЕСКИЙ СБОР НОВЫХ ВОПРОСОВ:**
  - Записывает вопросы, которых нет в базе
  - Пробует варианты пока не найдет правильный
  - Автоматически сохраняет с правильными ответами
  - Экспорт в JSON файл
- Думает 2-5 секунд перед каждым ответом
- Цель: 11 правильных ответов

**📖 Подробная инструкция:** [AUTO-QUIZ-COLLECTOR.md](./AUTO-QUIZ-COLLECTOR.md)

## ⚙️ Настройка

Все настройки в интерфейсе расширения:
- Задержки (минимум-максимум)
- Лимиты кликов
- Скорость прокрутки
- Включить/выключить тыквы

Настройки сохраняются автоматически.

## 🐛 Не работает?

1. Проверь что ты на mangabuff.ru
2. F12 → Console → посмотри ошибки
3. Перезагрузи расширение в chrome://extensions/
4. Перезагрузи страницу

## 📂 Что внутри

```
extension/               # Chrome расширение
├── manifest.json       # Конфиг
├── popup.html/css/js   # Интерфейс
├── content.js          # Основная логика
├── background.js       # Фоновый скрипт
└── icons/              # Иконки (создать)

mangabuff-auto-clicker.user.js  # Для Tampermonkey (только шахта)
console-script.js               # Для консоли (только шахта)
```

- Используй разумно, не злоупотребляй
